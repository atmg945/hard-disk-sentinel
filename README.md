## 硬盘坏之前，它早就知道了

很多人以为硬盘坏是突然发生的，其实不是。硬盘在"死"之前，往往会经历一段漫长的"亚健康期"——坏扇区慢慢增多、温度反复过高、SMART 参数悄悄报错。这些信号，肉眼看不见，Windows 自带工具也基本忽略。

Hard Disk Sentinel 做的就是盯住这些早期信号。它在后台安静运行，实时读取硬盘的 SMART 数据，一旦有异常，立刻通过邮件、声音、弹窗等方式通知你——给你留出足够时间把数据备份出去，而不是等到硬盘彻底挂掉才追悔莫及。

主界面用红黄绿三色直观展示健康等级，就算完全不懂 SMART 参数，一眼扫过去也知道自己的盘是不是"病入膏肓"。
